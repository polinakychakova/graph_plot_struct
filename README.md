# graph_plot_struct
Приложение для работы с матрицами смежности. Строится граф (от введенной матрицей в приложении, выбранной из файла или рандомно созданной самим приложением), а потом преобразовывается в граф связей этих вершин , чтобы на 1 уровне были основные вершины, а к последнему уровню вершины исходного графа сливались в одну, имитируя процесс сборки механизма
