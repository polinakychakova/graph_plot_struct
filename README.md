# graph_plot_struct
Приложение для работы с матрицами смежности. Строится исходный граф, и преобразовывается в граф связей этих вершин , чтобы на 1 уровне были основные вершины, а к последнему уровню вершины исходного графа сливались в одну, имитируя процесс сборки механизма

### Структура проекта
Проект состоит из следующих файлов:

  - menu.py — точка входа в приложение (основной интерфейс для пользователя);
  
  - check.py — содержит функции для проверки корректности входных данных; 
  
  - create.py — включает функции для создания матриц смежности (из файла или генерации случайных); 
  
  - graph_utils.py — набор утилит для работы с графами (построение, преобразование, анализ).
  
### Для запуска 
Используйте команду python menu.py

### Функционал
  1. Создание матриц смежности: из введенных пользоватлем данных, загруженного файла или генерация случайных матриц.
  
  2. Проверка данных: валидация входных данных (корректность матрицы, отсутствие ошибок).
  
  3. Преобразование и построение графов: построение заданного графа по матрице смежности и последующее его преобразование на грайфовое разложение(попарное слияние смежных вершин до тех пор, пока все вершины не "сольются" в одну). Пример выводимых данных представлен ниже.
  
  4. Интерфейс: простое меню для взаимодействия с пользователем (выбор операций, загрузка файлов, вывод результатов).
